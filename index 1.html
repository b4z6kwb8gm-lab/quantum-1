演算子の固有値方程式 固有値

対象の波動関数
$$\psi(x,t) = \exp\!\left[\frac{i}{\hbar}(px - Et)\right]$$
ア
位置演算子 $\hat{x}$ を $\psi(x,t)$ に作用させると？
$\hat{x}\,\psi(x,t) = \boxed{\text{ア}}\,\psi(x,t)$
ア = $x$（実数）
位置演算子は $\hat{x} = x$（掛け算演算子）。$\psi$ に作用すると $x\cdot\psi$ を返す。固有値は $x$ そのもの。
イ
運動量演算子 $\hat{p} = -i\hbar\dfrac{\partial}{\partial x}$ を $\psi$ に作用させると？
$\hat{p}\,\psi(x,t) = \boxed{\text{イ}}\,\psi(x,t)$
イ = $p$（運動量）
$\hat{p}\psi = -i\hbar \cdot \frac{i}{\hbar}p\,\psi = p\,\psi$。固有値は $p$。
ウ
自由粒子のハミルトニアン $\hat{H} = \dfrac{\hat{p}^2}{2m}$ を作用させると？
$\hat{H}\,\psi(x,t) = \boxed{\text{ウ}}\,\psi(x,t)$
ウ = $E = p^2/(2m)$
$\hat{H}\psi = \frac{p^2}{2m}\psi = E\,\psi$。固有値は全エネルギー $E$。
エ
上式の中で「固有値方程式」と呼ばれるのはどれか？（複数選択の問いだが、ここでは「当てはまるもの全てを含む選択肢」を選べ）
エ = b と c（イとウ）
固有値方程式とは「演算子×波動関数＝（定数）×波動関数」の形。(a) は $\hat{x}$ の固有値方程式にもなるが、量子力学で通常「固有値方程式」として重視されるのは 運動量・エネルギー固有値方程式 (b)(c)。特に (c) は時間依存しない Schrödinger 方程式そのもの。

📖 ステップバイステップ解説：演算子と固有値方程式

Step 1 ── 波動関数の確認

与えられた波動関数は「平面波」と呼ばれる形です：

$$\psi(x,t) = \exp\!\left[\frac{i}{\hbar}(px - Et)\right]$$

$p$ は運動量、$E$ はエネルギー、$\hbar = h/(2\pi)$ は換算プランク定数。この関数は空間方向に波数 $k = p/\hbar$、時間方向に角振動数 $\omega = E/\hbar$ で振動します。

Step 2 ── 位置演算子 (a)

量子力学では位置演算子を $\hat{x} = x$（変数そのものを掛ける演算子）と定義します：

$$\hat{x}\,\psi(x,t) = x\cdot\psi(x,t)$$
ポイント
固有値方程式の形 $\hat{A}\,\psi = a\,\psi$ において、$a$ が定数（$x$ に依存しない値）でなければ「固有値方程式」とは呼びません。位置演算子では固有値 $a = x$ となりますが、これは「$x$ という特定の値に対して」という意味で、粒子が位置 $x$ に確定した状態です。
Step 3 ── 運動量演算子 (b)

運動量演算子は $\hat{p} = -i\hbar\,\partial/\partial x$ と定義されます。$\psi$ を $x$ で偏微分してみましょう：

$$\frac{\partial \psi}{\partial x} = \frac{i}{\hbar}\,p\,\psi$$

これに $-i\hbar$ を掛けると：

$$\hat{p}\,\psi = -i\hbar \cdot \frac{i}{\hbar}\,p\,\psi = p\,\psi$$

固有値は $p$ です。この平面波は運動量 $p$ が確定した状態です。

Step 4 ── ハミルトニアン (c)：時間依存しない Schrödinger 方程式

自由粒子のハミルトニアムは $\hat{H} = \hat{p}^2/(2m)$。Step 3 の結果を使うと：

$$\hat{H}\,\psi = \frac{\hat{p}^2}{2m}\psi = \frac{p^2}{2m}\psi = E\,\psi \quad\left(E = \frac{p^2}{2m}\right)$$

これが時間依存しない Schrödinger 方程式（定常状態のシュレーディンガー方程式）です。

固有値方程式の定義
演算子 $\hat{A}$ に対し $\hat{A}\,\psi = a\,\psi$（$a$ は定数）が成り立つとき、この式を「固有値方程式」、$\psi$ を「固有関数」、$a$ を「固有値」と呼ぶ。(b)(c) は共に固有値方程式の形をとっている。

🔑 キーポイントまとめ

  • 位置演算子 $\hat{x} = x$：固有値 $= x$
  • 運動量演算子 $\hat{p} = -i\hbar\,\partial/\partial x$：固有値 $= p$
  • ハミルトニアン $\hat{H} = \hat{p}^2/(2m)+V$：固有値 $= E$
  • 固有値方程式 $\hat{A}\psi = a\psi$ の形をとるもの → 物理量が確定
  • 平面波は $\hat{p}$ と $\hat{H}$ の同時固有状態（$V=0$ の場合）

規格化因子 A, B, C 規格化

規格化条件
$$\int_{-\infty}^{\infty}|\psi(x)|^2\,dx = 1$$

（指定範囲以外はゼロ）

オ
$\psi(x) = A\sin\!\left(\dfrac{\pi x}{a}\right)$，$-a \le x \le a$ の規格化因子 $A$ は？
オ = $1/\sqrt{a}$
$\int_{-a}^{a}\sin^2\!\left(\frac{\pi x}{a}\right)dx = a$（sin² の積分は区間 $2a$ の半分）。よって $A^2 \cdot a = 1 \Rightarrow A = 1/\sqrt{a}$。
カ
$\psi(x) = B\cos\!\left(\dfrac{\pi x}{2a}\right)$，$-a \le x \le a$ の規格化因子 $B$ は？
カ = $1/\sqrt{a}$
$\int_{-a}^{a}\cos^2\!\left(\frac{\pi x}{2a}\right)dx = a$（cos² も同様）。$B = 1/\sqrt{a}$。
キ
$\psi(x) = Ce^{-|x|}$，$-\infty \le x \le \infty$ の規格化因子 $C$ は？
キ = $1/\sqrt{1} = 1$（より正確には $C = 1$）
$\int_{-\infty}^{\infty}e^{-2|x|}dx = 2\int_0^{\infty}e^{-2x}dx = 1$。よって $C^2 = 1 \Rightarrow C = 1$。

📖 ステップバイステップ解説：規格化因子の求め方

Step 1 ── 規格化とは何か

量子力学では $|\psi(x)|^2$ が「粒子が $x$ 付近にいる確率密度」です。全空間で積分すると確率の合計が 1 になるはずなので：

$$\int_{-\infty}^{\infty}|\psi(x)|^2\,dx = 1$$

規格化因子（$A, B, C$ など）はこの条件が満たされるように決める定数です。

Step 2 ── A の計算：$\sin^2$ の積分
$$\int_{-a}^{a}A^2\sin^2\!\left(\frac{\pi x}{a}\right)dx = 1$$

半角公式 $\sin^2\theta = (1-\cos 2\theta)/2$ を使います：

$$\int_{-a}^{a}\sin^2\!\left(\frac{\pi x}{a}\right)dx = \int_{-a}^{a}\frac{1-\cos\!\left(\frac{2\pi x}{a}\right)}{2}dx$$

第 2 項は一周期分の $\cos$ の積分でゼロ。第 1 項は $\frac{1}{2}\cdot 2a = a$。

$$A^2 \cdot a = 1 \implies A = \frac{1}{\sqrt{a}}$$
Step 3 ── B の計算：$\cos^2$ の積分
$$\int_{-a}^{a}\cos^2\!\left(\frac{\pi x}{2a}\right)dx$$

半角公式 $\cos^2\theta = (1+\cos 2\theta)/2$：

$$= \int_{-a}^{a}\frac{1+\cos\!\left(\frac{\pi x}{a}\right)}{2}dx = a + 0 = a$$

$\cos(\pi x/a)$ は $[-a,a]$ で一周期分なので積分はゼロ。よって $B = 1/\sqrt{a}$。

A と B が一致する理由
sin² と cos² は同じ積分値になります（$[-a,a]$ で半周期が異なるだけで面積は同じ）。
Step 4 ── C の計算：指数関数の積分

$|x|$ の絶対値があるので $x \ge 0$ と $x < 0$ に分けます：

$$\int_{-\infty}^{\infty}C^2 e^{-2|x|}dx = 2C^2\int_0^{\infty}e^{-2x}dx = 2C^2\cdot\frac{1}{2} = C^2$$

$C^2 = 1$ より $C = 1$（$C$ は正の実数）。

🔑 キーポイントまとめ

  • 規格化 ↔ $\int|\psi|^2 dx = 1$ から定数を逆算
  • $\int_{-L}^{L}\sin^2(n\pi x/L)\,dx = L$（半角公式で計算）
  • $\int_{-\infty}^{\infty}e^{-\alpha|x|}dx = 2/\alpha$（$\alpha > 0$）
  • 偶関数の積分は $2\times$ 片側積分でOK

1次元無限井戸型ポテンシャル (0 ≤ x ≤ L) 固有値問題

ポテンシャル
$$V(x) = \begin{cases} 0 & (0 \le x \le L) \\ \infty & (x > L,\; x < 0) \end{cases}$$
ク
$0 \le x \le L$ における時間依存しない Schrödinger 方程式は $\dfrac{d^2u}{dx^2} = \boxed{\text{ク}}\,u(x)$ と書ける。ク は？
ク = $-2mE/\hbar^2$（負の定数）
$\hat{H}u = Eu$、$V=0$ なので $-(\hbar^2/2m)d^2u/dx^2 = Eu$。整理して $d^2u/dx^2 = -(2mE/\hbar^2)u$。
ケ
境界条件（波動関数が井戸の壁でゼロになる）を式で表すと？
ケ = $u(0) = u(L) = 0$
無限ポテンシャルの壁では波動関数が完全に遮断されるため、境界で $u = 0$。
コ
最低エネルギー固有値（基底状態エネルギー）は？
コ = $E_1 = \dfrac{\pi^2\hbar^2}{2mL^2}$
$n=1$ のとき $k = \pi/L$、$E_1 = \hbar^2 k^2/(2m) = \pi^2\hbar^2/(2mL^2)$。
サ
次いで大きいエネルギー固有値（第1励起状態）は？
サ = $E_2 = 4E_1 = \dfrac{4\pi^2\hbar^2}{2mL^2} = \dfrac{2\pi^2\hbar^2}{mL^2}$
$n=2$ のとき $E_2 = 4E_1$。エネルギーは $n^2$ に比例する。

📖 ステップバイステップ解説：無限井戸型ポテンシャル

Step 1 ── 方程式の導出

$V=0$ の領域でのシュレーディンガー方程式：

$$-\frac{\hbar^2}{2m}\frac{d^2u}{dx^2} = Eu$$

両辺を $-\hbar^2/(2m)$ で割ると：

$$\frac{d^2u}{dx^2} = -\frac{2mE}{\hbar^2}\,u \equiv -k^2 u \quad \left(k = \frac{\sqrt{2mE}}{\hbar} > 0\right)$$
Step 2 ── 一般解

$-k^2 < 0$ なので一般解は三角関数：

$$u(x) = A\sin(kx) + B\cos(kx)$$
Step 3 ── 境界条件の適用

$u(0) = 0$：$B\cos(0) = B = 0$ → $B = 0$

$u(L) = 0$：$A\sin(kL) = 0$ → $kL = n\pi$（$n = 1, 2, 3, \ldots$）

$$k_n = \frac{n\pi}{L}$$
量子化の起源
$n$ が整数に限定されることで、エネルギーが離散化（量子化）されます。これが「量子力学」の核心。
Step 4 ── エネルギー固有値
$$E_n = \frac{\hbar^2 k_n^2}{2m} = \frac{n^2\pi^2\hbar^2}{2mL^2}, \quad n = 1, 2, 3, \ldots$$

最低エネルギー（$n=1$）：

$$E_1 = \frac{\pi^2\hbar^2}{2mL^2}$$

第 1 励起状態（$n=2$）：$E_2 = 4E_1 = 4 \times \dfrac{\pi^2\hbar^2}{2mL^2}$

Step 5 ── 固有関数の形
$$u_n(x) = \sqrt{\frac{2}{L}}\sin\!\left(\frac{n\pi x}{L}\right), \quad 0 \le x \le L$$

$\sqrt{2/L}$ は規格化因子。$n$ が増えるほど波の「節」が増え、エネルギーが高くなります。

🔑 キーポイントまとめ

  • $V=0$ 領域の方程式：$d^2u/dx^2 = -k^2 u$（$k = \sqrt{2mE}/\hbar$）
  • 境界条件 $u(0)=u(L)=0$ → $k_n = n\pi/L$（量子化）
  • エネルギー $E_n = n^2\pi^2\hbar^2/(2mL^2) \propto n^2$
  • 基底状態でもエネルギーゼロにならない（零点エネルギー）
  • 固有関数 $\propto \sin(n\pi x/L)$ は $n-1$ 個の節を持つ

対称無限井戸 (|x| ≤ L) 偶関数・奇関数

ポテンシャル
$$V(x) = \begin{cases} 0 & (|x| \le L) \\ \infty & (|x| > L) \end{cases}$$
ア
$|x| \le L$ における時間依存しない Schrödinger 方程式の右辺係数は？
$\dfrac{d^2u}{dx^2} = \text{ア}\cdot u(x)$
ア = $-2mE/\hbar^2$
Tab [3] と同形。$V=0$ なので同じ式。
イ
境界条件は？
イ = $u(-L) = u(L) = 0$
壁は $x = \pm L$ にある。
ウ
偶関数解の最低エネルギー固有値 $E_{\min}^{(\text{偶})}$ は？
ウ = $\dfrac{\pi^2\hbar^2}{8mL^2}$
偶関数解は $u \propto \cos(k_n x)$、$u(\pm L)=0$ → $k_n L = (2n-1)\pi/2$（奇数半整数倍）。最小は $n=1$：$k_1 = \pi/(2L)$、$E = \hbar^2k_1^2/(2m) = \pi^2\hbar^2/(8mL^2)$。
エ
奇関数解の最低エネルギー固有値 $E_{\min}^{(\text{奇})}$ は？
エ = $\dfrac{\pi^2\hbar^2}{2mL^2}$
奇関数解は $u \propto \sin(k_n x)$、$u(\pm L)=0$ → $k_n = n\pi/L$。最小は $n=1$：$E = \pi^2\hbar^2/(2mL^2)$。
オ
全固有値のうち最も低いもの（基底状態）は？
オ = $\dfrac{\pi^2\hbar^2}{8mL^2}$（偶関数解）
$\pi^2\hbar^2/(8mL^2) < \pi^2\hbar^2/(2mL^2)$ なので偶関数解の方が低い。
カ
次いで大きいエネルギー準位は？
カ = $\dfrac{\pi^2\hbar^2}{2mL^2}$（奇関数解の最低、$\sin(\pi x/L)$）
次は $E_{\min}^{(\text{奇})} = \pi^2\hbar^2/(2mL^2) = 4 \times (\pi^2\hbar^2/8mL^2)$。

📖 ステップバイステップ解説：対称ポテンシャルの偶奇分類

Step 1 ── なぜ偶関数・奇関数に分類できるか

ポテンシャルが $V(x) = V(-x)$（原点対称）のとき、ハミルトニアンは空間反転に対して不変です。よって固有関数は偶関数か奇関数のどちらかにすることができます（パリティが良い量子数になる）。

偶関数・奇関数の定義
偶関数：$u(-x) = +u(x)$　奇関数：$u(-x) = -u(x)$
Step 2 ── 偶関数解（コサイン型）

一般解 $u = A\sin(kx) + B\cos(kx)$ のうち、偶関数成分は $u = B\cos(kx)$。

境界条件 $u(\pm L) = 0$：

$$\cos(kL) = 0 \implies kL = \frac{(2n-1)\pi}{2}, \quad n = 1, 2, 3, \ldots$$

$n=1$：$k_1 = \pi/(2L)$、エネルギー：

$$E_{\min}^{(\text{偶})} = \frac{\hbar^2 k_1^2}{2m} = \frac{\pi^2\hbar^2}{8mL^2}$$
Step 3 ── 奇関数解（サイン型）

奇関数成分は $u = A\sin(kx)$。境界条件 $u(\pm L) = 0$：

$$\sin(kL) = 0 \implies kL = n\pi, \quad n = 1, 2, 3, \ldots$$

$n=1$：$k_1 = \pi/L$、エネルギー：

$$E_{\min}^{(\text{奇})} = \frac{\pi^2\hbar^2}{2mL^2}$$
Step 4 ── 全エネルギー準位の並び

両系列を合わせると（$E_0 \equiv \pi^2\hbar^2/(8mL^2)$ とおく）：

$$E_0,\; 4E_0,\; 9E_0,\; 16E_0, \ldots$$

これは $[-L,L]$ 幅 $2L$ の井戸の固有値 $E_n = n^2\pi^2\hbar^2/(2m\cdot(2L)^2) = n^2E_0$ と一致します。

Step 5 ── [3] と [4] の比較

[3] では幅 $L$（$0 \le x \le L$）、[4] では幅 $2L$（$-L \le x \le L$）。基底エネルギーは幅の 2 乗に反比例するので、[4] は [3] の $1/4$ になります。

🔑 キーポイントまとめ

  • 原点対称ポテンシャル → 固有関数は偶関数か奇関数に分類可能
  • 偶関数解：$\cos(kx)$、$kL = (2n-1)\pi/2$
  • 奇関数解：$\sin(kx)$、$kL = n\pi$
  • 最低エネルギーは偶関数解（$n=1$ の $\cos$ 解）
  • 井戸の幅 $W$ → $E_1 \propto 1/W^2$

変数分離と積分 変数分離

問1　積分の計算

問1(a)
$\displaystyle\int_{-\pi}^{\pi}\cos(2x)\sin(3x)\,dx = ?$
答え = 0
積和公式：$\cos(2x)\sin(3x) = \frac{1}{2}[\sin(5x)+\sin(x)]$。$\sin$ は奇関数なので $[-\pi,\pi]$ での積分はゼロ。（直交性による）
問1(b)
$\displaystyle\int_{-\pi}^{\pi}\cos^2(4x)\,dx = ?$
答え = $\pi$
$\cos^2(4x) = (1+\cos(8x))/2$。$\int_{-\pi}^{\pi}1/2\,dx = \pi$、$\cos(8x)$ の積分はゼロ（4周期分）。

問2　変数分離法

Schrödinger 方程式と演算子
$$\hat{H}\psi(x,t) = \hat{E}\psi(x,t), \quad \hat{H} = -\frac{\hbar^2}{2m}\frac{\partial^2}{\partial x^2} + V(x), \quad \hat{E} = i\hbar\frac{\partial}{\partial t}$$

$\psi(x,t) = X(x)T(t)$ と書けるとする

問2(a)–1
$\psi = X(x)T(t)$ を代入して $t$ のみの式と $x$ のみの式に分離したとき、時間部分の方程式の形は？
答え = $i\hbar\,\dot{T}/T = E$（定数）
両辺を $XT$ で割ると左辺は $x$ のみ、右辺は $t$ のみ → 両辺とも定数 $E$。
問2(b)–T解
$t$ に関する微分方程式 $i\hbar\,dT/dt = ET$ の一般解は？
答え = $T(t) = C\,e^{-iEt/\hbar}$
$dT/T = -iE/\hbar\,dt$ を積分。時間振動因子は $e^{-i\omega t}$（$\omega = E/\hbar$）の形。
問2(b)–X解
$V(x)=0$ のとき、$x$ に関する方程式の一般解は？
答え = $X(x) = Ae^{ikx} + Be^{-ikx}$（$k = \sqrt{2mE}/\hbar$）
$d^2X/dx^2 = -k^2X$ の一般解。右進行波と左進行波の重ね合わせ。

📖 ステップバイステップ解説：変数分離法と三角関数の直交性

Step 1 ── 三角関数の直交性（問1）

フーリエ解析の基礎となる直交性：

$$\int_{-\pi}^{\pi}\cos(mx)\cos(nx)\,dx = \pi\,\delta_{mn}$$ $$\int_{-\pi}^{\pi}\sin(mx)\sin(nx)\,dx = \pi\,\delta_{mn}$$ $$\int_{-\pi}^{\pi}\cos(mx)\sin(nx)\,dx = 0 \quad (\text{常に})$$

最後の式は「cos（偶関数）×sin（奇関数）= 奇関数」の積分がゼロであることからも分かります。

Step 2 ── $\cos^2$ の積分（問1b）
$$\int_{-\pi}^{\pi}\cos^2(4x)\,dx = \int_{-\pi}^{\pi}\frac{1+\cos(8x)}{2}dx = \frac{2\pi}{2} + 0 = \pi$$

$\cos(8x)$ は $[-\pi,\pi]$ でちょうど 4 周期なので積分はゼロ。

Step 3 ── 変数分離の手順（問2a）

$\psi = X(x)T(t)$ を $i\hbar\,\partial\psi/\partial t = \hat{H}\psi$ に代入：

$$i\hbar\,X\dot{T} = \hat{H}[X]\cdot T$$

両辺を $XT$ で割る：

$$\underbrace{\frac{i\hbar\,\dot{T}}{T}}_{\text{$t$ のみ}} = \underbrace{\frac{\hat{H}[X]}{X}}_{\text{$x$ のみ}} = E$$

左辺は $t$ だけ、右辺は $x$ だけの関数が等しいので、両辺は共通の定数 $E$ でなければならない。

Step 4 ── 時間方程式の解（問2b）
$$i\hbar\frac{dT}{dt} = ET \implies \frac{dT}{T} = \frac{-iE}{\hbar}dt \implies T(t) = e^{-iEt/\hbar}$$

これが波動関数の時間振動部分。エネルギー $E$ が大きいほど速く振動する。

Step 5 ── 空間方程式の解（$V=0$）（問2b）
$$\frac{d^2X}{dx^2} = -k^2 X, \quad k = \frac{\sqrt{2mE}}{\hbar}$$

一般解：$X(x) = Ae^{ikx} + Be^{-ikx}$ または $C\sin(kx) + D\cos(kx)$（同値）。

平面波との接続
$T(t) \cdot X(x) = e^{-iEt/\hbar}(Ae^{ikx}+Be^{-ikx})$ の $A$ 成分は $e^{i(kx-Et/\hbar)}$。これが [1] の平面波 $e^{i(px-Et)/\hbar}$（$p = \hbar k$）に対応！

🔑 キーポイントまとめ

  • cos×sin = 奇関数 → $[-\pi,\pi]$ での積分はゼロ（直交性）
  • $\cos^2(nx)$ の積分 → 半角公式で定数項と $\cos(2nx)$ に分解
  • 変数分離 $\psi = X(x)T(t)$ → 両辺を定数 $E$ で等値
  • 時間解：$T = e^{-iEt/\hbar}$（位相振動）
